# Sanitized Regression Trace Examples

These snippets are copied from the regression logs and cleaned for documentation.

Use this page when the README summary is not enough and you want to see what each test actually generated, what the reference model expected, and what the DUT did.

## Log Signals

| Tag | Meaning |
|---|---|
| `PRELOAD STEP` | The sequence generated this instruction and wrote it into instruction memory. |
| `REF STEP` | The reference model executed this instruction and calculated the expected state. |
| `DUT FETCH_*` | RTL instruction fetch request/response. |
| `DUT WRITEBACK` | RTL register writeback. |
| `DUT LOAD_*` | RTL data load request/response. |
| `DUT STORE_*` | RTL data store request/response. |
| `DUT HALT` | RTL halted on the intentional illegal instruction. |
